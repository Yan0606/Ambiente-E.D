Listas Duplamente Encadeadas

Uma lista duplamente encadeada, também conhecida como lista duplamente ligada, é uma estrutura de dados em que cada elemento (nó) contém um ponteiro para o próximo nó e um ponteiro para o nó anterior. Dessa forma, é possível percorrer a lista tanto no sentido da frente para trás quanto de trás para frente.

Exemplo

Aqui está um exemplo de implementação de uma lista duplamente encadeada em C#:

public class No
{
   public int Data;
   public No Prox;
   public No Pont;

    public No(int data)
   {
       Data = data;
       Prox = null;
       Pont = null;
    }
}

Nesse exemplo, temos uma classe que chamamos do No. A classe No representa um nó da lista e possui três membros: Data (para armazenar o valor), Prox (ponteiro para o próximo nó) e Pont (ponteiro para o nó anterior).

public class ListaDupla
{
   private No inicio;
   private No fim;
   public void Add(int data);
{
    No newNo = new No(data;)
   if (inicio == null)
    {
       inicio = newNo;
       fim = newNo;
    }
    else
    }
        fim.Prox = newNo;
        newNo.Pont = fim;
        fim = newNo;
    }
}

Criamos uma classe que chama ListaDupla, é responsável por gerenciar a lista. Ela possui um ponteiro para o início da lista e um ponteiro para o final da lista. O método Add adiciona um novo elemento no final da lista, atualizando corretamente os ponteiros.

public void MostrarAvancar()
{
   No current = inicio;
   while (current != null)
   {
        Console.Write(current.Data + " ");
        current = current.Prox;
    }
   Console.WriteLine();
}
    public void MostrarRecuar()
    {
        No currente = fim;
        while (currente != null)
    {
       Console.Write(current.Data + " ");
        current = current.Pont;
    }
    Console.WriteLine();
}

Os métodos MostrarAvancar e MostrarRecuar percorrem a lista, imprimindo os elementos na ordem direta e reversa, respectivamente.

public class Program
{
    public static void Main(string[] args)
    {
       ListaDupla list = new ListaDupla();

       list.Add(1);
       list.Add(2);
       list.Add(3);
       list.Add(4);

       Console.WriteLine("Avançar: ";)
       list.MostrarAvancar();

       consoleWriteLine("Recuar:");
       list.MostrarRecuar();
    }
}
