Pilha Encadeada

Uma pilha encadeada, também conhecida como pilha ligada ou stack, é uma estrutura de dados linear que segue o princípio do "last in, first out" (LIFO), o que significa que o último elemento adicionado à pilha é o primeiro a ser removido. Neste tipo de estrutura, os elementos são organizados em uma sequência encadeada de nós, onde cada nó contém um valor e uma referência para o próximo nó na pilha.

Exemplo

Aqui está um exemplo de implementação de uma pilha encadeada em C#:

Definindo a Classe PilhaEncadeada:
  • A classe PilhaEncadeada é definida para representar a pilha encadeada.
  • Ela contém uma classe interna chamada No, que representa os nós individuais da pilha.
  • Há uma variável privada topo que aponta para o nó no topo da pilha.

using System;

class PilhaEncadeada
{
    private class No
    {
       public int Valor { get; set; }
       public No Proximo { get; set; }

      public No(int valor)
       {
          Valor = valor;
          Proximo = null;
       }
    }

    private No topo;

    public PilhaEncadeada()
    {
       topo = null;
    }

Construtor da Pilha:

  • O construtor da classe PilhaEncadeada inicializa o topo como nulo, indicando que a pilha está vazia no início.

Método Push:

  • O método Push é usado para adicionar elementos à pilha.
  • Ele cria um novo nó com o valor especificado e o torna o novo nó no topo da pilha, ajustando as referências.

Método Pop:

  • O método Pop é usado para remover e retornar o elemento no topo da pilha.
  • Ele verifica se a pilha está vazia (o topo é nulo) e lança uma exceção se estiver.
  • Caso contrário, ele obtém o valor do topo, ajusta o topo para apontar para o próximo nó e retorna o
  • valor removido.

Método Print:

  • O método Print é usado para exibir os elementos da pilha na ordem em que foram empilhados.
  • Ele percorre a pilha a partir do topo e imprime os valores dos nós.

public void Push(int valor)
{
    No novoNo = new No(valor);
    novoNo.Proximo = topo;
   topo = novoNo;
}

public int Pop()
{
    if (topo == null)
    {
       throw new InvalidOperationException("A pilha está vazia.");
    }

    int valorRemovido = topo.Valor;
    topo = topo.Proximo;
    return valorRemovido;
}

public void Print()
{
    No atual = topo;
    Console.Write("Pilha: ");
    while (atual != null)
    {
       Console.Write(atual.Valor + " ");
       atual = atual.Proximo;
    }
       Console.WriteLine();
    }
}

Classe Program - Método Main:
  • No método Main, criamos uma instância da classe PilhaEncadeada chamada pilha.
Empilhando Valores:
  • Chamamos o método Push quatro vezes para adicionar os valores 1, 2, 3 e 4 à pilha, respectivamente .
Impressão da Pilha:
  • Chamamos o método Print para imprimir o conteúdo da pilha, que será "Pilha: 4 3 2 1", pois os elementos são impressos na ordem em que foram empilhados.
Desempilhando um Valor:
  • Chamamos o método Pop para remover o elemento no topo da pilha (4, no caso) e armazenamos esse valor em valorRemovido.

Imprimindo o Valor Removido:
  • Imprimimos o valor removido usando Console.WriteLine, que mostrará "Valor removido: 4".

Empilhando o Valor 10:
  • Chamamos o método Push para adicionar o valor 10 à pilha.

Impressão Atualizada da Pilha:
  • Chamamos o método Print novamente para imprimir o conteúdo atualizado da pilha, que será "Pilha: 10 3 2 1".

class Program
{
    static void Main()
    {
       PilhaEncadeada pilha = new PilhaEncadeada();

       pilha.Push(1);
       pilha.Push(2);
       pilha.Push(3);
      pilha.Push(4);

       pilha.Print(); // Pilha: 4 3 2 1

       int valorRemovido = pilha.Pop(); // Remove o 4
      Console.WriteLine("Valor removido: " + valorRemovido);

      pilha.Push(10); // Adiciona o 10

      pilha.Print(); // Pilha: 10 3 2 1
    }
}
