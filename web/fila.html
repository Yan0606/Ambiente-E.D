Fila Encadeada FIFO

Uma fila encadeada FIFO (First-In-First-Out) é uma estrutura de dados que funciona como uma fila da vida real, onde o primeiro elemento adicionado é o primeiro a ser removido. A seguir vou mostrar como criar, adicionar e remover um nó de uma fila encadeada.

Exemplo

Crie uma classe, crie o "dado" para armazenar o nó e referencie o nó para o proximo da fila

using System;
   public class NoFila
   {
      public int Dado;     // O dado armazenado no nó
      public NoFila Proximo;  // Referência para o próximo nó na fila

    public NoFila(int dado)
    {
      Dado = dado;
       Proximo = null;
    }
}

Defina a classe para a fila encadeada.

public class FilaEncadeada
{
    private NoFila primeiro; // O primeiro nó na fila
    private NoFila ultimo; // O último nó na fila

    // Construtor para criar uma fila vazia
    public FilaEncadeada()
    {
        primeiro = null;
        ultimo = null;
    }

Método para adicionar um nó na lista

public void Enqueue(int dado)
{
    NoFila novoNo = new NoFila(dado);

    // Se a fila estiver vazia, o novo nó será o primeiro e o último
    if (primeiro == null)
    {
        primeiro = novoNo;
        ultimo = novoNo;
    }
    else
    {
        // Caso contrário, o novo nó é adicionado ao final da fila
        ultimo.Proximo = novoNo;
    }
}

Método para remover e retornar o elemento da fila

public int Dequeue()
{
    // Verifique se a fila está vazia
   if (primeiro == null)
    {
       throw new InvalidOperationException("A fila está vazia.");
    }


    // Obtenha o dado do primeiro nó
    int dadoRemovido = primeiro.Dado;

    // Atualize o primeiro nó para o próximo nó na fila
   primeiro = primeiro.Proximo;

    // Se a fila ficou vazia, atualize o último nó também
    if (primeiro == null)
    {
       ultimo = null;
    }

    return dadoRemovido;
    }
}

Na Main de seu projeto insira:

class Program
{
    static void Main()
    {
       FilaEncadeada fila = new FilaEncadeada();

       // Adicione elementos à fila
       fila.Enqueue(1);
       fila.Enqueue(2);
      fila.Enqueue(3);

       // Remova e imprima os elementos da fila
      Console.WriteLine("Elementos da fila:");

       while (true)
    {
       try
       {
          int elemento = fila.Dequeue();
          Console.WriteLine(elemento);
       }
       catch (InvalidOperationException)
       {
          Console.WriteLine("A fila está vazia.");
          break;
          }
       }
    }
}

Neste exemplo, a classe NoFila representa um nó na fila com um campo de dado e uma referência para o próximo nó na fila. A classe FilaEncadeada mantém referências para o primeiro e o último nó na fila e fornece métodos Enqueue para adicionar elementos à fila e Dequeue para remover elementos da fila seguindo a política FIFO. O programa principal demonstra como usar a fila encadeada.
