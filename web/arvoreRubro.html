Árvore Rubro-Negra

Uma árvore rubro-negra é outra árvore binária de busca balanceada. As árvores rubro-negras são implementadas usando uma técnica chamada cor. Cada nó de uma árvore rubro-negra tem uma cor, que pode ser vermelha ou preta.

As árvores rubro-negras são mais eficientes que as árvores AVL para operações de inserção e remoção. Isso ocorre porque as árvores rubro-negras usam uma técnica de balanceamento mais simples, que é mais eficiente para essas operações.

Exemplo de Inserção

Note que sempre que um elemento que está sendo adicionado é maior que o elemento raiz, ele sempre será redirecionado a direita, e menor para a esquerda, nessa operação a tendência é sempre deixar o maior para cima, por isso ao inserir o “26” e o “54”, há trocas de posições e giros.

Exemplo de Remoção

Veja que a operação é semelhante a inserção, ele busca o elemento a ser excluído através de comparações de > ou <, mantendo sempre o padrão do maior elemento ir para direita e o menor para esquerda.

Exemplo de Busca

Perceba que o elemento desejado a ser buscado irá percorrer normalmente como a inserção e remoção.
