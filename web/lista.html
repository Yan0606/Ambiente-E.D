Listas Simplesmente Encadeadas

Uma lista simplesmente encadeada em C# é uma estrutura de dados que consiste em nós ligados sequencialmente, onde cada nó contém um valor e uma referência para o próximo nó da lista. A primeira referência da lista é chamada de cabeça (head) e a última referência é nula, indicando o final da lista.

Exemplo

A seguir iremos mostrar um exemplo básico de uma implementação de uma lista simplesmente encadeada, na linguagem de programação C#

public class No
{
   public int valor;
   public No Prox;

    public No(int Valor)
   {
       this.valor = Valor;
       this.noProx = null;
       Pont = null;
    }

Neste exemplo, a classe No representa um nó da lista, com uma propriedade valor para armazenar o valor do nó e uma propriedade noProx para apontar para o próximo nó.

public class lista
{
   public No inicio;
   public No fim;

    public lista()
   {
       this.inicio = null;
       this.fim = null;
    }

A classe Lista é responsável por gerenciar a lista. Ela possui um campo inicio que representa a cabeça da lista e um campo fim que representa a calda da lista.

public void inserirIncicio(int value)
{
   No novoNo = new(value)

   if(this.incio == null && this.fim == null){
       this.incio = novoNo;
       this.fim = novoNo;
       Console.WriteLine("Insira a posição: " + this.incio.valor;)
   }
   else if (this.incio != null){
        novoNo.noProx = this.incio;
       this.incio = novoNo;
      Console.WriteLine("Insira a posição: " + this.incio.valor);
   }

Vamos inserir um Nó no inicio da lista, primeiro criamos o método inserir inicio, então verificamos se a lista está fazia, se ele estiver fazia o o novo nó sera inserido, dessa forma, ele será tanto a cabeça quanto a calda da lista.

Mas se a lista não estiver fazia, devemos definir que o proximo do nosso nó sera o atual cabeça da lista "this.inicio".

public void percurso(){

   No noAux = this.incio;
   Console.Write("\n[ Lista: ");
   while(noAux != null){
      Console.Write(noAux.valor);
      if(noAux.noProx != null){
         Console.Write(" > ");
      }
      noAux = noAux.noProx;
   }
   Console.Write("]\n\n");
}

Agora adicionamos o método percurso para para percorrer a lista e imprimir os valores

namespace Lista{

   class Program {
      static void Main(string[] args){

         lista lista = new();
         lista.inserirInicio(22);
         lista.inserirInicio(12);
         lista.inserirInicio(7);

          lista.percurso();
      }
   }
}

No método Main, criamos instâncias da classe lista, adicionamos alguns valores à lista e em seguida imprimimos os valores. No exemplo acima, a saída será: 7 12 22, que são os valores adicionados à lista.
